# .NET和C#有什么区别

答：.Net一般指.Net Framework框架，是Microsoft为开发应用程序而创建的一个具有革命意义的平台

C#：运行在.NET(CLR)上的，用于创建应用程序的一种高级编程语言。

# 一列数的规则如下: 1、1、2、3、5、8、13、21、34...... 求第30位数是多少，用递归算法实现

```c#
public static int Foo(int i)
{
    if (i <= 0)
    {
        return 0;
    }
    else if (i > 0 && i <= 2)
    {
        return 1;
    }
    else
    {
        return Foo(i - 1) + Foo(i - 2);
    }
}
```
