# 单选题

1. 阅读下面代码

```c#
	class Program
    {
		public static void ThrowException()
        {
			throw new Exception();
        }
    
		static void Main(string[] args)
        {
			try
            {
				Console.WriteLine("try");
                ThrowException();
            }
			catch
            {
				Console.WriteLine("catch");
            }
			finally
            {
				Console.WriteLine("finally");
            }
			Console.Read();
        }
    }
```

请问代码运行结果是：( A )

A.   try / catch / finally

B.    try 

C.    try / catch

D.   try / finally

:pencil:异常处理的执行步骤
